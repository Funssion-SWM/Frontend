## 핵심 기능

#### 1. 컨텐츠 생산성 향상을 위한 기능

- 생성형 AI를 활용한 텍스트 자동 생성
- 생성형 AI를 활용한 요약, 태그 자동 생성
- 오픈소스(https://github.com/steven-tey/novel)를 활용한 블럭 기반 웹에디터
- 시리즈

#### 2. 컨텐츠 생산 동기 부여를 위한 기능

- Post history
- 등급 제도
- 채용 도우미 서비스

#### 3. 일반 커뮤니티 기능

- 댓글
- 알림
- 검색
- 좋아요
- Q&A
